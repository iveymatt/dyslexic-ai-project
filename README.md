## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
# Clone the repository
git clone https://github.com/iveymatt/dyslexic-ai-project.git
cd dyslexic-ai-project

# Install dependencies
cd frontend
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Our Philosophy

> "Neurodivergent brains aren't broken. They just work differently. Stop trying to 'fix' us. Start building tools that work WITH how we think."

We believe:
- Dyslexia is a language-processing difference, not a disability
- Lateral thinking (connections, exploration) is just as valid as linear thinking
- ADHD brains need both structure AND flexibility
- Autistic communication styles are valid and valuable
- Masking is exhausting and shouldn't be required
- AI tools should adapt to US, not the other way around
